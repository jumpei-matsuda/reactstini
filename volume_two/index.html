SCSS

まずは「SCSS」というCSSの拡張言語をインストールしましょう。SCSSはネスト構造が使え、視覚的にHTMLと似た感覚でコーディングできることが特徴の言語です。ちなみに似た言語で「SASS」もありますが、SCSSではネスト構造を「{ }」を使用して表現することに対して、SASSではそれを使用しないで記述できるという違いがあります。

SASS

.content
  display: flex
  flex-direction: row
    p
      text-align: center

SCSS

.content {
  display: flex;
  flex-direction: row;
    p {
      text-align: center;
    }
}

個人的には「{ }」がある方が階層が明確でわかりやすいと感じるため、今回はSCSSを使用していきます。以下のコマンドからインストールしましょう。（sassをインストールしていますが、これでscssも使用できます。） 

yarn add sass

また今回はSCSSに加えて「CSS Modules」を使用していきます。

プロジェクトが大きくなるにつれて、CSSのclass名が被ったり、それを回避するためにやたら長かったり直感的にわかりづらい名前を付けざるを得ないという状況に陥ったことはないでしょうか？「CSS Module」は各モジュールごとにcssファイルを作成し、上記のような問題を回避することができる技術です。

簡単に説明すると、バンドル時にファイルごとに固有の識別子を付与することで、ファイル間でclass名の重複を避けることができるといった仕組みです。実際にブラウザ上で開発者ツールを開いて確認すると（F12）、画像のようにclass名の末尾に識別子が付与されていることが確認できると思います。

MUI & emotion & date-fns

「MUI」はコンポーネントライブラリの一つで、デザインや一定の機能が実装されたコンポーネントをimportできるようになります。「emotion」はCSSライブラリで、MUIのコンポーネントのスタイリングで使用されているため同時にインストールします。
また、今回はスケジュール管理のアプリを作成するため、「@mui/x-date-pickers」と、Date型を制御する「date-fns」も同時にインストールしていきます。

以下のコマンドを実行し、vite.contig.tsとtsconfig.jsonを編集しましょう。 

yarn add @mui/material @emotion/react @emotion/styled
yarn add -D @emotion/babel-plugin
yarn add @mui/x-date-pickers
yarn add date-fns

vite.contig.ts

import { defineConfig } from 'vite';
import react from '@vitejs/plugin-react';
import tsconfigPaths from 'vite-tsconfig-paths';

// https://vitejs.dev/config/
export default defineConfig({
  plugins: [
    react({
      jsxImportSource: '@emotion/react',
      babel: {
        plugins: ['@emotion/babel-plugin'],
      },
    }),
    tsconfigPaths()
  ]
});

tsconfig.json

{

TopPage.tsx

import { useState } from 'react'
import {
  Box,
  Button,
  Table,
  TableBody,
  TableCell,
  TableContainer,
  TableHead,
  TableRow,
  TextField,
  Typography
} from '@mui/material'
import { format } from 'date-fns'
import { DateTimePicker } from '@mui/x-date-pickers/DateTimePicker'
import styles from './TopPage.module.scss'

type EventResouces = {
  id: string
  title: string
  start: string
  end: string
}

const TopPage: React.FC = () => {
  const defaultEventResouces: EventResouces[] = [
    { id: '1', title: 'event 1', start: '2023-01-01 08:00', end: '2023-01-01 12:00' },
    { id: '2', title: 'event 2', start: '2023-01-05 12:00', end: '2023-01-05 15:00' }
  ]
  const dateFormat = 'yyyy-MM-dd hh:mm'

  const [title, setTitle] = useState('')
  const [start, setStart] = useState<Date | null>(null)
  const [end, setEnd] = useState<Date | null>(null)
  const [eventResouces, setEventResouces] = useState<EventResouces[]>(defaultEventResouces)

  const onRegister = () => {
    if (!title || !start || !end) {
      alert('タイトル、開始時刻、終了時刻は入力必須です')

      return
    }

    setEventResouces([
      ...eventResouces,
      {
        id: (eventResouces.length + 1).toString(),
        title,
        start: format(start, dateFormat),
        end: format(end, dateFormat)
      }
    ])

    setTitle('')
    setStart(null)
    setEnd(null)
  }

  return (
    <Box className={styles['content']}>
      <Typography variant='h1' className={styles['title']}>Top Page</Typography>
      <Box className={styles['form-content']} >
        <Box className={styles['form-input']}>
          <TextField
            label='タイトル'
            onChange={(e) => setTitle(e.target.value)}
            value={title}
          />
          <DateTimePicker
            inputFormat={dateFormat}
            onChange={(value) => setStart(value)}
            renderInput={(params) => <TextField {...params} label='開始時刻' />}
            value={start}
          />
          <DateTimePicker
            inputFormat={dateFormat}
            onChange={(value) => setEnd(value)}
            renderInput={(params) => <TextField {...params} label='終了時刻' />}
            value={end}
          />
        </Box>
        <Button
          className={styles['form-button']}
          onClick={onRegister}
          variant='contained'
        >
          登録
        </Button>
      </Box>

      <TableContainer>
        <Table>
          <TableHead>
            <TableRow>
              <TableCell>タイトル</TableCell>
              <TableCell>開始時刻</TableCell>
              <TableCell>終了時刻</TableCell>
            </TableRow>
          </TableHead>
          <TableBody>
            {eventResouces.map((item) => (
              <TableRow key={item.id} hover>
                <TableCell>{item.title}</TableCell>
                <TableCell>{item.start}</TableCell>
                <TableCell>{item.end}</TableCell>
              </TableRow>
            ))}
          </TableBody>
        </Table>
      </TableContainer>
    </Box>
  )
}

export default TopPage

FullCalendar

最後に「FullCalendar」というライブラリをインストールしましょう。こちらのライブラリは、さまざまなレイアウトのカレンダーを実装することが可能です。
では以下のコマンドを実行しましょう。 

 yarn add @fullcalendar/core @fullcalendar/react @fullcalendar/daygrid @fullcalendar/interaction @fullcalendar/timegrid

インストールが完了したら、以下のファイルを修正しましょう。
