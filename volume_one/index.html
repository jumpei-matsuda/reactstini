Viteでプロジェクトを作成する

まずはReactプロジェクトの骨組みを作成していきます。Reactは、JavaでいうSpringのようなフレームワークとは違い、あくまでもJavascriptのライブラリという位置づけのため、サードパーティが提供するプロジェクトを作成・運用するツールを使用します。その中で今回は「Vite」というプロダクトを使用してきます。
では早速プロジェクトを作成しましょう。コマンドプロンプトでプロジェクトを作成したいディレクトリまで移動し、以下のコマンドを実行しましょう。 

yarn create vite reactstini --template=react-ts

「reactstini」の部分はプロジェクト名なので、任意で変更して頂いて大丈夫です。続いて以下のコマンドを実行しましょう。 

cd reactstini
yarn

「yarn」コマンドを実行することで、プロジェクト内に作成された「package.json」内の「dependencies」と「devDependencies」に記載されたライブラリをインストールすることができます。それでは骨組みが完成したので、以下のコマンドを実行しReactプロジェクトを起動させましょう。 

yarn dev

「http://127.0.0.1:5173」をクリックするとブラウザ上で開くことができます。

ちなみにコマンドプロンプトはVSCodeで「ターミナル(T) > 新しいターミナル > reactstini」の手順から開くことができるため、開発中はこちらを活用しましょう。
以上でプロジェクトの作成は完了です。


ESlintのインストールと初期設定

では続いて「ESLint」を導入していきましょう。ESLintはコードチェック・解析をしてバグを生みにくい適切なコードの書き方になるよう指摘したり、ときには修正してくれるツールです。今回はESLint本体に加え、Typescriptにも対応したパッケージもインストールしていきます。

まずは本体のインストールです。 

yarn add -D eslint

また「.eslintignore」 ファイルを作ってESLintチェックの対象外となるファイルを定義しておきましょう。現状では一行だけで大丈夫です。 

vite.config.ts

プロジェクトのESLintの設定は以上です。次ににVSCodeでのコーディング中にもESLintが走るように拡張機能を追加しましょう。VSCodeの画面左のツールバーから「拡張機能」をクリックし、ESLintを検索しインストールしましょう。

インストールが完了したら、ツールバーから「管理 > コマンドパレット」をクリック、「基本設定：設定(JSON) を開く」から「setting.json」を開き、以下を追加しましょう。 

{
  "editor.codeActionsOnSave": {
    "source.fixAll.eslint": true
  },
  "editor.formatOnSave": false,
  "eslint.packageManager": "yarn",
  "typescript.enablePromptUseWorkspaceTsdk": true,
}

最後に個々のファイルだけではなく、全体にまとめてESLintを走らせるコマンドを package.jsonに追加しましょう。 

"scripts": {
  "dev": "vite",
  "build": "tsc && vite build",
  "preview": "vite preview",
  "lint": "eslint src/**/*.{js,jsx,ts,tsx}", // 追加
  "lint:fix": "eslint --fix src/**/*.{js,jsx,ts,tsx}", // 追加
  "preinstall": "npx typesync || :" // 追加
},

創刊号は以上です。ここまで完了すればReact開発も快適に進めていけるかと思います。

またこの状態のプロジェクトを雛形としてGit等で管理することをおすすめします。新規プロジェクトを立ち上げる際はクローンを作成して「yarn」コマンドを実行するだけで開発環境が整うため、先ほどの設定の手間を削減できます。
