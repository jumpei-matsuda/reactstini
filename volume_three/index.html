FullCalendarのレイアウトを変更

まずFullCalendarのレイアウトを変更していきましょう。今のままでは学習時間が一目で判断することができないので、横軸に一週間の日付、縦軸に時間が表示されるカレンダーにします。FullCalendarではprosの「plugins」の配列にプラグインを追加することでさまざまなカレンダーの表示に切り替えることができるので、TopPage.tsxを以下のように変更してみましょう。

TopPage.tsx

import { useState } from 'react'
import { EventInput } from '@fullcalendar/core'
import jaLocale from '@fullcalendar/core/locales/ja'
import FullCalendar from '@fullcalendar/react'
import timeGridPlugin from '@fullcalendar/timegrid'
import { Box, Button, TextField, Typography } from '@mui/material'
import { DateTimePicker } from '@mui/x-date-pickers/DateTimePicker'
import styles from './TopPage.module.scss'

const TopPage: React.FC = () => {
  const defaultEventResouces: EventInput[] = [
    { id: '1', title: 'event 1', start: '2023-01-01 08:00', end: '2023-01-01 12:00' },
    { id: '2', title: 'event 2', start: '2023-01-05 12:00', end: '2023-01-05 15:00' }
  ]
  const dateFormat = 'yyyy-MM-dd hh:mm'

  const [title, setTitle] = useState('')
  const [start, setStart] = useState<Date | null>(null)
  const [end, setEnd] = useState<Date | null>(null)
  const [eventResouces, setEventResouces] = useState<EventInput[]>(defaultEventResouces)

  const onRegister = () => {
    if (!title || !start || !end) {
      alert('タイトル、開始時刻、終了時刻は入力必須です')

      return
    }

    setEventResouces([
      ...eventResouces,
      {
        id: (eventResouces.length + 1).toString(),
        title,
        start,
        end
      }
    ])

    setTitle('')
    setStart(null)
    setEnd(null)
  }

  return (
    <Box className={styles['content']}>
      <Typography variant='h1' className={styles['title']}>Top Page</Typography>
      <Box className={styles['form-content']} >
        <Box className={styles['form-input']}>
          <TextField
            label='タイトル'
            onChange={(e) => setTitle(e.target.value)}
            value={title}
          />
          <DateTimePicker
            inputFormat={dateFormat}
            onChange={(value) => setStart(value)}
            renderInput={(params) => <TextField {...params} label='開始時刻' />}
            value={start}
          />

            inputFormat={dateFormat}
            onChange={(value) => setEnd(value)}
            renderInput={(params) => <TextField {...params} label='終了時刻' />}

        </Box>
        <Button
          className={styles['form-button']}
          onClick={onRegister}
          variant='contained'
        >
          登録
        </Button>
      </Box>

      <FullCalendar
        events={eventResouces}
        initialView="timeGridWeek"
        locale='ja'
        locales={[jaLocale]}
        plugins={[timeGridPlugin]}
      />
    </Box>
  )
}

export default TopPage

イベント追加機能の実装

FullCalendarの全体的なレイアウトが固まったところで、続いてイベント追加の機能を実装していきましょう。時間単位のバーを選択するとモーダルが表示され、そこでイベント名を設定してイベントを登録できるようにしていきます。

Contextの実装

今回は簡単なアプリ実装なので状態管理等のライブラリはインストールせず、Contextを使用してグローバルに状態管理を行っていきます。FullCalendarのpropsに渡すオブジェクトをグローバルに管理するので、src配下に「stores」ディレクトリ、その配下に「eventResources」ディレクトリを作成し、以下のファイルを作成しましょう。

stores/eventResources/EventResourcesProvider

import { createContext, useContext, useState } from 'react';
import { EventInput } from '@fullcalendar/core';

type UseProviderEventResources = {
  eventResouces: EventInput[]
  addEventResouce: (values: EventInput) => void
  editEventResouce: (values: EventInput) => void
  deleteEventResource: (id: string) => void
}

const initialState: UseProviderEventResources = {
  eventResouces: [],
  addEventResouce: () => {},
  editEventResouce: () => {},
  deleteEventResource: () => {}
};

/**
 * コンテクストオブジェクト
 */
export const eventContext = createContext(initialState);

/**
 * コンテクストオブジェクトの現在値を返すフック
 */
export const useEventResources = (): UseProviderEventResources => useContext(eventContext);

export const useProviderEventResources = (): UseProviderEventResources => {
  const defaultValue: EventInput[] = [
    { id: '1', title: 'event 1', start: '2023-01-01 08:00:00', end: '2023-01-01 12:00' },
    { id: '2', title: 'event 2', start: '2023-01-05 12:00:00', end: '2023-01-05 15:00' }
  ]
  const [eventResouces, setEventResources] = useState<EventInput[]>(defaultValue);

  const addEventResouce = (event: EventInput) => {
    setEventResources([
      ...eventResouces, {
        ...event
      }
    ])
  }

  const editEventResouce = (event: EventInput) => {
    setEventResources([
      ...eventResouces.map((item) => item.id === event.id ? event : item)
    ])
  }

  const deleteEventResource = (id: string) => {
    setEventResources([
      ...eventResouces.filter(item => item.id !== id)
    ])
  }

  return {
    eventResouces,
    addEventResouce,
    editEventResouce,
    deleteEventResource
  }
}

ここではeventsオブジェクトと、そのeventを登録する関数を定義し、画面では「useEventResources」から、そのeventsオブジェクトと登録関数を呼び出すことができるようになってます。以下のファイルを修正して、実際にTop画面から呼び出してみましょう。

EditEventModalContent.tsx

import { useState } from 'react';
import { EventImpl } from '@fullcalendar/core/internal';
import { Button, Dialog, TextField, Typography } from '@mui/material';
import { DateTimePicker } from '@mui/x-date-pickers/DateTimePicker';
import styles from './EditEventModal.module.scss';
import { useEventResources } from '@/stores/eventResources/useEventResources';

type Props = {
  isOpen: boolean
  registeredEvent: EventImpl | undefined
  onClose: () => void
};

const EditEventModalContent: React.FC<Props> = ({ isOpen, registeredEvent, onClose }) => {
  const [title, setTitle] = useState(registeredEvent?.title);
  const [start, setStart] = useState(registeredEvent?.start ?? null);
  const [end, setEnd] = useState(registeredEvent?.end ?? null);
  const { editEventResouce, deleteEventResource } = useEventResources();

  const onRegisterButtonClicked = () => {
    if (!start || !end) {
      alert('開始日または終了日を入力してください。');

      return;
    }

    editEventResouce({
      ...registeredEvent,
      id: registeredEvent?.id,
      title,
      start,
      end
    });
    onClose();
  };

  const onDeleteButtonClicked = (id?: string) => {
    if (!id) {
      alert('対象のデータが存在しません');

      return;
    }

    const confirmFlg = confirm('削除してもよろしいですか？');

    if (confirmFlg) {
      deleteEventResource(id);
      onClose();
    }
  };

  return (
    <Dialog open={isOpen}>
      <div className={styles['content']}>
        <Typography variant='h5' align='center'>イベントを編集</Typography>
        <div className={styles['start-end-time']}>
          <label className={styles['title-input']} htmlFor='title'>
            タイトル：
            <TextField name='title' value={title} onChange={(e) => setTitle(e.currentTarget.value)} />
          </label>
          <label className={styles['title-input']} htmlFor='title'>
            開始時刻：
            <DateTimePicker
              value={start}
              onChange={(value) => setStart(value)}
              renderInput={(params) => <TextField {...params} />}
              inputFormat={'yyyy/MM/dd hh:mm'}
            />
          </label>
          <label className={styles['title-input']} htmlFor='title'>
            終了時刻：
            <DateTimePicker
              value={end}
              onChange={(value) => setEnd(value)}
              renderInput={(params) => <TextField {...params} />}
              inputFormat={'yyyy/MM/dd hh:mm'}
            />
          </label>
        </div>
        <div className={styles['close-button']}>
          <Button variant='contained' onClick={() => onRegisterButtonClicked()}>更新</Button>
          <Button variant='contained' color="error" onClick={() => onDeleteButtonClicked(registeredEvent?.id)}>削除</Button>
          <Button variant='outlined' onClick={() => onClose()}>閉じる</Button>
        </div>
      </div>
    </Dialog>
  );
};

import jaLocale from '@fullcalendar/core/locales/ja';
import dayGridPlugin from '@fullcalendar/daygrid';
import interactionPlugin from '@fullcalendar/interaction';
import FullCalendar from '@fullcalendar/react';
import timeGridPlugin from '@fullcalendar/timegrid';
import useAddEventModal from './AddEventModal/useAddEventModal';
import useEditEventModal from './EditEventModal/useEditEventModal';
import styles from './TopPage.module.scss';
import { useEventResources } from '@/stores/eventResources/useEventResources';

const TopPage: React.FC = () => {
  const { eventResouces } = useEventResources();
  const { AddEventModal, open: openAddModal } = useAddEventModal();
  const { EditEventModal, open: openEditModal } = useEditEventModal()

  return (
    <div className={styles['content']}>
      <AddEventModal />
      <EditEventModal />
      <h1>Top Page</h1>
      <FullCalendar
        headerToolbar={{
          left: 'title timeGridWeek dayGridMonth',
          right: 'today prev,next'
        }}
        buttonText={{
          week: 'week',
          month: 'month'
        }}
        events={eventResouces}
        initialView="timeGridWeek"
        locale='ja'
        locales={[jaLocale]}
        plugins={[timeGridPlugin, dayGridPlugin, interactionPlugin]}
        scrollTime={null}
        select={(arg) => openAddModal(arg)}
        selectable
        eventClick={(arg) => openEditModal(arg.event)}
      />
    </div>
  );
};

export default TopPage;

ざっくりと内容を説明します。AddとEditではほぼ同じ動きをしているので、Addを中心に説明していきます。少し複雑になってくるため動かしながら理解を深めてみてください。

まず「useAddEventModal.tsx」は、コンポーネントの「AddEventModal」、「AddEventModal」を開くための「open」という関数、「AddEventModal」を閉じるための「close」という関数を返却するカスタムフックです。「TopPage.tsx」にて「useAddEventModal」から上記の3つを呼び出すことができるような構成にしています。

「open」では「selectedTime」という、先ほどカレンダーの修正で確認した時間情報のオブジェクト（変数名：selectedTime）を引数にとり、「AddEventModal」コンポーネント内で管理しているstateにセットすることで、モーダル内で選択した時間を管理できるようにしています。

「close」では実行時に管理していたstateをクリアします。

そして肝心の「AddEventModalContent.tsx」ですが、openで「AddEventModal」に設定された「selectedTime」をそのままpropsとして受け取り、さらに「start」「end」に分割してstateで管理しています。「start」「end」はDatePickerのvalueに設定するため変更が可能な状態にしています。

登録ボタン押下時、本号前半で実装したコンテキストの「useEventResources」から呼び出し可能な「addEventResouces」を実行し、コンテキスト内にデータを登録することができます。TopPageではコンテキストからeventオブジェクトを取得しているので、登録と共にTopPageにも追加したeventが反映されています。

以上がモーダルの解説です。実際の現場では「AddEventModalContent.tsx」内の「title」「start」「end」などのフォーム情報はライブラリで管理したり、モーダルの共通コンポーネントを作成し、ロジック部分のみ独自で実装するなど簡略かつメンテナンス性が高くなるように実装されるので、もし時間がある方は工夫して見てください。
